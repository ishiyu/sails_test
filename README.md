# sails_test
Sails + Vue.js + bower で遊んでみる

# サーバの起動方法
node.js がインストールされている必要があります。

```
$ npm -g install bower
$ npm -g install sails

$ git clone http://github.com/ishiyu/sails_test
$ cd sails_test
$ bower install
$ sails lift
```

# 確認方法
http://localhost:1337/chat へアクセスするとチャット画面が見れます。
２つのブラウザを用意し、メッセージを入力するとリアルタイムに反映されることが確認できます。
